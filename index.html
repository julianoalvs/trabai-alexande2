circulo cromatico

Bom o Circulo Cromatico é uma Ferramenta usada para representar de forma simples e intuitiva as cores visiveis pelo olho humano ele é muito usado geralmente para a mistura de cores

Harmonias Cromaticas

Harmonias Cromaticas: normalmente são usadas para criar combinações as principais harmonias são:

  • Monocromática: é uma combinação de cores que usa uma unica cor
  • Complementar: é uma combinação que usa cores opostas no ciruclo cromatico
  • Análoga: é uma combinação de cores que utiliza cores que estão lado a lado no circulo cromatico
  • Triádica: é uma combinação de cores que estão equidistantes no circulo cromatico
  • Complementar Dividida: é uma combinção de cores que usa uma cor principal e duas adjacentes do circulo cromatico
  • Quádrupla: é um acombinação de cores que combina 4 cores equisdistantes no criculo cromatico

Exemplos de harmonias Cromaticas
